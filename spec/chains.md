# コンビニチェーン仕様

プレイヤーが選択可能な3つのコンビニチェーン。

---

## ローサン

### 特徴
- 唯一バリエーション展開ができる
- 他2社よりも特性は弱い

### バリエーション
| タイプ | 説明 |
|--------|------|
| 単価が低いローサン | 低価格帯店舗 |
| 単価が高いローサン | 高価格帯店舗 |
| 普通のローサン | 標準店舗 |
| ちょっといいローサン | 中上位店舗 |

### 名物
牛乳がうまい

---

## ファモマ

### 特徴
- 怪音波を出して住民を洗脳する
- ZOCとドミナントエリアが広い
- コンビニ単価が高い

### トレードオフ
広いZOC・ドミナントエリア ⇔ 高いコンビニ建設コスト

### 名物
チキンがうまい

---

## セバンイレバン

### 特徴
- コンビニ単価が低くコスパがいい
- ドミナントエリア広がるにつれて経営が適当になる（上げ底効果）
- ステータスが徐々に下がっていく

### 特殊能力: イレバン
7社以上でドミナントすると **イレバン** する

### トレードオフ
低コスト ⇔ ドミナント拡大によるステータス低下
