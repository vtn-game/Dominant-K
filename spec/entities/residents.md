# 住民定義

## 共通パラメータ

| パラメータ名 | 型 | 説明 |
|-------------|-----|------|
| `id` | string | 一意識別子 |
| `displayName` | string | 表示名 |
| `moveSpeed` | float | 移動速度 |
| `spendingPower` | float | 購買力（収益への寄与度） |
| `faithResistance` | float | 洗脳耐性（高いほど信仰度が上がりにくい） |
| `spawnWeight` | int | 出現重み（人口比率） |

---

## サラリーマン (Salaryman)

通勤途中にコンビニに立ち寄る。購買力が高い。

### ステータス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `moveSpeed` | 1.2 | 急いでいる |
| `spendingPower` | 1.5 | 高い |
| `faithResistance` | 0.8 | やや低い |
| `spawnWeight` | 30 | 多い |

### 特性
- 朝(7-9時)・夕(17-20時)に出現率UP
- 弁当・飲料を優先購入

---

## 学生 (Student)

時間はあるが金はない。

### ステータス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `moveSpeed` | 1.0 | 標準 |
| `spendingPower` | 0.6 | 低い |
| `faithResistance` | 0.5 | 洗脳されやすい |
| `spawnWeight` | 25 | やや多い |

### 特性
- 昼(11-14時)・放課後(15-18時)に出現率UP
- 一度洗脳されると他チェーンに移りにくい

---

## 主婦/主夫 (Homemaker)

日用品の購入が多い。コスパ重視。

### ステータス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `moveSpeed` | 0.8 | ゆっくり |
| `spendingPower` | 1.2 | やや高い |
| `faithResistance` | 1.2 | やや高い |
| `spawnWeight` | 20 | 標準 |

### 特性
- 昼間(10-16時)に出現率UP
- 価格の安いチェーンを優先（セブンイレバン優遇）

---

## 高齢者 (Elder)

ゆっくり歩くが、一度信じたら変わらない。

### ステータス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `moveSpeed` | 0.5 | 遅い |
| `spendingPower` | 0.8 | やや低い |
| `faithResistance` | 1.5 | 高い |
| `spawnWeight` | 10 | 少ない |

### 特性
- 早朝(6-8時)に出現率UP
- 信仰度が一度MAXになると固定（離反しない）

---

## ITエンジニア (Engineer)

深夜のコンビニ常連。エナジードリンクを買う。

### ステータス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `moveSpeed` | 0.9 | やや遅い |
| `spendingPower` | 1.3 | やや高い |
| `faithResistance` | 1.0 | 標準 |
| `spawnWeight` | 10 | 少ない |

### 特性
- 深夜(22-翌4時)に出現率UP
- 怪音波への耐性+20%（ヘッドホンしてる）

---

## インド系住民 (IndianResident)

N-KXi（西葛西モチーフ）の特徴的な住民層。

### ステータス

| パラメータ | 値 | 備考 |
|-----------|-----|------|
| `moveSpeed` | 1.0 | 標準 |
| `spendingPower` | 1.0 | 標準 |
| `faithResistance` | 0.7 | やや低い |
| `spawnWeight` | 5 | 少ない |

### 特性
- 全時間帯で均等に出現
- 同種の住民が近くにいると信仰度上昇率+50%（コミュニティ効果）

---
